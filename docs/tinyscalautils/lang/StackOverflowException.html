StackOverflowException

class StackOverflowException(error: StackOverflowError) extends RuntimeException

Stack overflows as exceptions instead of errors.

Constructor:

Builds a new exception with the same message, cause and stack trace as the error. Note that error is ''not'' used as cause. See tinyscalautils.control.limitedStack
